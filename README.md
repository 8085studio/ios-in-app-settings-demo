# iOS In-app Settings Demo

## 系統需求: CocoaPods

### 安裝
```
$ sudo gem install cocoapods
```

## 將InAppSettingsKit導入專案

`開啟Terminal，移至專案目錄下`
執行
```
$ pod install
```

## 開啟專案
改以`InAppSettingsDemo.xcworkspace`檔來開啟專案
